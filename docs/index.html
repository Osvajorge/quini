Cómo funciona

3 pasos. Sin trucos. Todo el código y datos son públicos.

Paso 1

Modelo estadístico

Detecta el edge

Te decimos cuánto apostar usando Kelly Criterion para maximizar tu banca a largo plazo sin arriesgar de más.
